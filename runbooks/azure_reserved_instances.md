# Azure Reserved Instances and Savings Plans

## What Are Reserved Instances
Reserved Instances let you commit to 1-year or 3-year term for Azure VMs and SQL Databases
in exchange for up to 72% discount over pay-as-you-go pricing.

## When to Use
- Workloads running 24/7 like production databases and always-on APIs
- Predictable stable resource consumption
- Services: Virtual Machines, Azure SQL Database, Cosmos DB, App Service, AKS nodes

## Savings Estimates
- Virtual Machines 1-Year: 30-40% savings
- Virtual Machines 3-Year: 55-65% savings
- Azure SQL DB 1-Year: 25-35% savings
- Azure SQL DB 3-Year: 50-60% savings

Azure Portal go to Cost Management and Billing then Reservations then Add
